# 🛒 Speed Delivery

Speed Delivery is a high-performance e-commerce web application built with Next.js 15 (App Router), TypeScript, Tailwind CSS, and the Shopify Storefront API. It offers a modern, responsive shopping experience with features like persistent cart state, dynamic collections, product variants, and a complete checkout flow.

## 🚀 Features

- 🔍 **Dynamic Search** – Fast product and collection search powered by Shopify GraphQL API.
- 🧾 **Product Detail Page** – With image gallery, variants, quantity selector, and buy now.
- 🛍️ **Cart System** – Redux-powered cart with persistent state, quantity management, and live totals.
- 💳 **Shopify Checkout Integration** – Seamlessly redirects to Shopify-hosted checkout.
- 🖼️ **Skeleton UI Loaders** – Skeleton loaders for product and collection pages to improve perceived performance.
- 📱 **Responsive Design** – Fully mobile-optimized using Tailwind CSS.
- 🧩 **Reusable Hooks** – Modular hooks for cart, checkout, and wishlist management.
- 📦 **Collections Navigation** – Dynamic collections dropdown in header using Shopify categories.
- ⚡ **PWA Ready** – Optimized for performance and ready to ship as a Progressive Web App.
- 🌐 **Deployed on Vercel** – Instant builds, preview deployments, and global CDN.

## 🧑‍💻 Tech Stack

- **Frontend**: Next.js 15 (App Router), TypeScript, Tailwind CSS
- **State Management**: Redux Toolkit
- **API Integration**: Shopify Storefront GraphQL API
- **Deployment**: Vercel
